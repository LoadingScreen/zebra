About ZebraSign

What is ZebraSign?

ZebraSign is a program for creating and verifying ring signatures. This website provides verification, but not signing.

What are Ring Signatures?

A ring signature is a digital signature produced using one private key plus some number of public keys. The resulting message can be proven to have been signed by exactly one of those keyholders, but there is no way to prove which one. This is different from Group Signatures.

How to Use This Website

If you want to verify a ring signature, you can do that here. Copy and paste the entire ring-signed message into the text box, and it will tell you if the identities listed were in fact used to sign the message.
